Запрос:
GET /albums/21
Ответ:
{
  "album": 21,
  "title": "Greatest Hits",
  "artist": "The Beatles",
  "release_date": "1970-11-10",
  "genre": "Rock"
}
Описание:
Получить информацию о альбоме по его id

Запрос:
GET /albums
Описание:
возвращает список всех альбомов

Запрос:
POST /albums
{
  "title": "New Album",
  "artist_id": 15,
  "release_date": "2023-09-15",
  "genre_id": 8
}
Описание:
создать новый альбом

Запрос:
PUT /albums/21
{
  "title": "Updated Album Title",
  "release_date": "2023-10-01"
}
Описание:
обновить информацию о существующем альбоме

Запрос:
DELETE /albums/21
Описание:
удаляет альбом с указанным id

*** C Остальными все тоже самое! ***

Исполнители (Artists):

Получение списка всех исполнителей:
GET /artists

Получение информации о конкретном исполнителе:
GET /artists/{artist_id}

Создание нового исполнителя:
POST /artists

Редактирование информации об исполнителе:
PUT /artists/{artist_id}

Удаление исполнителя:
DELETE /artists/{artist_id}
